# stock-portfolio-tracker
A real-time stock P&L and Target price calculator built with Python, yfinance and tkinter.
User Input --
Stocks Symbol defined in NSE which are mostly normal stocks name other we can find on internet easily.
Number of stocks.
Buying price or Target price.
Purpose fullfill of user--
1) User can check real time profit and profit percentage.
2) If user want to know how much percantage need or amount from CMP. So he will get idea instead of check & calculate.
3) If user stuck in any stock and want to sell cost to cost so he will get idea how much need from CMP. Excpect how much till will get.
